7.3 电气安装工程
---------------------------

7.3.2 动力配电柜和成套柜
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

.. raw:: html

 <p style="text-align:justify"><a href="#idt7.3.2.2"> 7.3.2.2</a> <span id="idt7.3.2.2"> 接地关系到人身、设备安全，一般装有电器的可开启的盘、柜门采用裸铜软线与盘、柜的金属框架连接接地，当门上的电器绝缘损坏时，可以避免门上带电，危及操作人员的人身安全。</span></p>

7.3.3 变压器
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

.. raw:: html

 <p style="text-align:justify"><a href="#idt7.3.3.2"> 7.3.3.2</a> <span id="idt7.3.3.2"> 变压器低压侧中心点与接地装置的接地干线直接连接是最近的接地路径,可以确保运行中人身和电气设备的安全。</span></p>
 <p style="text-align:justify"><a href="#idt7.3.3.6"> 7.3.3.6</a> <span id="idt7.3.3.6"> 气体继电器是油浸变压器的保护继电器之一，装在变压器箱体与油枕的连接水平段中间。当变压器过载或局部故障时，使线圈有机绝缘或变压器油发生气化，形成的气体会升到箱体顶部。为有利气体流向气体继电器发出报警信号，并使气体经油枕泄放，因此要有升高坡度，不允许倒置。</span></p>

7.3.4 低压电器
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

.. raw:: html

 <p style="text-align:justify"><a href="#idt7.3.4.4"> 7.3.4.4</a> <span id="idt7.3.4.4"> 电气间隙是指空气中裸露电体间或裸露导电体与接地间的最小净距离。根据现行国家标准《电气装置安装工程低压电器施工及验收规范》（GB 50254-96）的规定，500V 以下“不同相母线最小电气间隙”为 10 mm。考虑到港口地区气候潮湿、空气中盐雾高的特殊情况，在征求多方意见后，本标准规定“不同相母线最小电气间隙”为 12 mm。</span></p>
 
7.3.8 低压电器
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

.. raw:: html

 <p style="text-align:justify"><a href="#idt7.3.8.3"> 7.3.8.3</a> <span id="idt7.3.8.3"> 为了规范接线，确保安全用电，螺口灯具和照明开关在安装时要求相线位置一致，也可给维修人员提供安全操作保障。如果位置紊乱，检修时没有切断相线，容易产生触电事故。</span></p>
 <p style="text-align:justify"><a href="#idt7.3.8.4"> 7.3.8.4</a> <span id="idt7.3.8.4"> 在三相五线制中，零线和保护接地线不能混用，除在变压器中心点可互连外，其余各点均不能相互连通。插座的接线位置要严格区分，否则有可能导致线路工作不正常和危及人身安全。此条文是根据现行行业标准《港口设备安装工程技术规范》（JTJ 280-2002 ）的有关规定和现行国家标准《建筑电气工程施工质量验收规范》（GB 50303-2002）的有关规定编写的。</span></p>

7.3.11 线路敷设
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

.. raw:: html

 <p style="text-align:justify"><a href="#idt7.3.11.4"> 7.3.11.4</a> <span id="idt7.3.11.4"> 目前电缆终端头和中间接头的产品种类很多，不同产品的电缆头在材质、工艺、技术要求、检测手段等各个方面的差异也很大，所以电缆头的制作安装要符合产品技术文件的规定。</span></p>

7.3.12 硬母线、滑接线与软电缆
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

.. raw:: html

 <p style="text-align:justify"><a href="#idt7.3.12.4"> 7.3.12.4</a> <span id="idt7.3.12.4"> 在母线搭接面上不能涂中性凡士林，而只能涂电力复合脂。中性凡士林的滴点太低，只有 54℃，在正常的运行温度 70℃的情况下，早已流淌，使母线接头间产生间隙，侵入灰尘、水分，增加了母线接头的接触电阻，引起接头发热，而电力复合脂的滴点可高达 180℃～220℃，可以避免上述问题。</span></p>

7.5 轨道式起重装卸设备安装工程
-------------------------------------

7.5.3 大车运行机构
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

.. raw:: html

 <p style="text-align:justify"><a href="#idt7.5.3.1"> 7.5.3.1</a> <span id="idt7.5.3.1"> 防风装置是港口起重装卸设备为防止突发性阵风和台风引起滑移和倾覆而设置的一种安全保护装置。抗滑移安全装置主要有铁鞋（也称防风楔块）、锚定装置、防爬器、夹轮器、刚性连接拉杆等型式；防倾覆安全装置主要有防风拉索、防风系固锁销等型式。防风装置的连接形式主要是焊接或高强度螺栓连接。</span></p>

7.5.12 机内电梯
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

.. raw:: html

 <p style="text-align:justify"><a href="#idt7.5.12.1"> 7.5.12.1</a> <span id="idt7.5.12.1"> 由于电梯涉及到人身安全，国家对电梯的安装有特别严格规定。电梯主要设备如驱动主机、轿箱、门系统、安全装置等一般均由电梯制造厂专业人员安装调试，故本条只对机内电梯导轨和塔架等有可能由非电梯专业人员安装的钢结构安装误差做出了规定。</span></p>

7.5.14 称量装置
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

.. raw:: html

 <p style="text-align:justify"><a href="#idt7.5.14.1"> 7.5.14.1</a> <span id="idt7.5.14.1"> 称量装置系指料斗秤、电子皮带秤、小车秤及其他用于对装卸物料作商业和管理计量的装置。由于其型式和称量原理各不相同，因此规定他们的检验要求应满足设计要求和符合产品技术文件的规定。</span></p>

7.5.18 安全装置
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

.. raw:: html

 <p style="text-align:justify"><a href="#idt7.5.18.1"> 7.5.18.1</a> <span id="idt7.5.18.1"> 安全装置关系到设备、人身和货物安全，是企业安全生产、设备正常运行和提高生产效率的重要保证。现行国家标准、行业标准或是设计文件，都对各种安全保护装置的设置和性能提出了严格要求，因此，本标准制定时，对于安全装置的设置和检验也给予了特别的重视，在内容上也有了比较详细的充实。</span></p>
 <p style="text-align:justify"><a href="#idt7.5.18.2"> 7.5.18.2</a> <span id="idt7.5.18.2"> 机内电梯安全保护装置一般包括上下限位开关、限速器、安全钳、缓冲器、门锁装置和电气保护装置等。</span></p>

7.11 设备试运行
---------------------------

.. raw:: html

 <p style="text-align:justify"><a href="#idt7.11.0.1"> 7.11.0.1</a> <span id="idt7.11.0.1"> 供配电系统通常由高电压、大容量设备组成，一旦发生事故，损失严重，因此在供电系统试运行前需严格检验。设备空载试运转前的静态检验，就是为保证试运转的顺利进行而作充分准备。</span></p>
 <p style="text-align:justify"><a href="#idt7.11.0.2"> 7.11.0.2</a> <span id="idt7.11.0.2"> 变配电系统一旦投入运行，断电检修很困难，因此本条对试运行检验提出了详细要求，以保证设备运行后的安全性和可靠性。</span></p>
 <p style="text-align:justify"><a href="#idt7.11.0.17"> 7.11.0.17</a> <span id="idt7.11.0.17"> 空载联动试运转是单机设备进行的无负荷试车和系统进行的无负荷联动试车，并通过调整达到设计技术要求的试验性运转。</span></p>
 <p style="text-align:justify"><a href="#idt7.11.0.19"> 7.11.0.19</a> <span id="idt7.11.0.19"> 重载试运转是设备和系统在空载联动试运转并经过交工验收后，按照设计要求进行的试验性运转。</span></p>
